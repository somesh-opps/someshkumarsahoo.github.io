## Website Details

This website includes:

- A fixed top navigation bar for quick section navigation
- A hero and introduction section with profile image and short bio
- About Me section
- Education and experience highlights
- Events attended and certificates
- Membership and completed courses
- Contact section with email, phone, and social links
- Back-to-top button for smoother navigation

---

## Tech Stack

| Technology | Purpose |
| --- | --- |
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, and visual design |
| JavaScript (Vanilla JS) | Navigation interactions and smooth scrolling |
| Remix Icon CDN | Social and UI icons |

---

## Project Structure

- `index.html` - Main website layout and content
- `style.css` - Styling for the complete website
- `script.js` - Smooth scrolling and section navigation behavior
- `scriptForContact.js` - Contact and social link interactions

---

## Run Locally

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
